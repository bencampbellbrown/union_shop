This folder should contain Noto font files used by the app.

Please download appropriate fonts and place them here with the exact filenames referenced in `pubspec.yaml`:

- assets/fonts/NotoSans-Regular.ttf
- assets/fonts/NotoSans-Bold.ttf
- assets/fonts/NotoColorEmoji.ttf

Suggested sources:
- NotoSans: https://fonts.google.com/specimen/Noto+Sans
- Noto Color Emoji: https://github.com/googlefonts/noto-emoji or https://fonts.google.com/specimen/Noto+Color+Emoji

After adding these files, run:

flutter pub get
